
# ChatApp With java Socket



A simple Java-based chat application with real-time messaging using socket programming. Multiple clients can connect to the server and communicate with each other.
## Technology Used
- Socket Programming
- Javas
- Multithreading
- GUI design


## How To Run ChatApp W/O GUI



1. Compile
- javac ChatServer.java ChatClient.java ChatLauncher.java
2. Start the Server
- java ChatServer
3. Start Clients
- java ChatClient
## How to Run with GUI
- No need any compilation, Just run the code on NetBeans!
